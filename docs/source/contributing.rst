Hacking on HTTPretty
====================

install development dependencies
--------------------------------


.. note:: HTTPretty uses `GNU Make
          <https://www.gnu.org/software/make/>`_ as default build
          tool.


.. code:: bash

    make dependencies


next steps
----------

1. run the tests with make:

.. code:: bash

   make tests

2. hack at will
3. commit, push etc
4. send a pull request
